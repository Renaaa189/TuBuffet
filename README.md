# TuBuffet
El proyecto surgió a partir de una problemática común en los entornos escolares: las largas filas y los tiempos de espera en los buffets/kioscos, especialmente durante los recreos, donde la demanda es alta y el tiempo limitado. Esta situación no solo genera incomodidad en alumnos y profesores, sino que también dificulta la organización y el control para quienes gestionan el servicio.
En base a esta problemática, se pensó en una solución que aproveche la tecnología para simplificar el proceso de compra. Así nace la idea de una aplicación que permita realizar pedidos de forma virtual, rápida y sencilla, evitando esperas innecesarias y mejorando la experiencia general dentro del ámbito escolar.
Al mismo tiempo, el proyecto busca aportar una herramienta útil para los vendedores, facilitando la gestión del inventario y contribuyendo a una mayor eficiencia en las ventas. Como valor agregado, se incorpora una billetera virtual individual,la cual promueve la educación financiera en niños y jóvenes, fomentando el uso responsable y organizado del dinero desde edades tempranas.
