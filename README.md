## Descripción

El proyecto surge a partir de una problemática común en entornos escolares: las largas filas y los tiempos de espera en los buffets/kioscos durante los recreos.

Esta situación genera incomodidad en alumnos y docentes, y dificulta la organización para quienes gestionan el servicio.

Como solución, se propone una aplicación móvil que permite realizar pedidos de forma virtual, rápida y organizada, evitando esperas innecesarias y mejorando la experiencia general.

Además, el sistema incorpora una billetera virtual individual que fomenta la educación financiera, promoviendo el uso responsable del dinero desde edades tempranas.

---

## Objetivo del sistema

- Reducir tiempos de espera en el buffet  
- Mejorar la organización de pedidos  
- Facilitar la gestión de ventas e inventario  
- Brindar control y visibilidad a padres/tutores  
- Incorporar herramientas educativas (billetera virtual)  

---

## <img src="https://media2.giphy.com/media/QssGEmpkyEOhBCb7e1/giphy.gif" width="25"> Funcionalidades principales

### Gestión de pedidos
- Compra anticipada desde el celular  
- Menú organizado por categorías (bebidas, golosinas, almuerzos)  
- Sistema de “compra rápida” basado en historial  

### Sistema económico
- Billetera virtual individual  
- Historial de pedidos  
- Sistema de puntos y recompensas  
- Descuentos según institución  

### Experiencia de usuario
- Interfaz simple e intuitiva  
- Acceso mediante código institucional  
- Flujo rápido y claro de compra  

### Administración
- Creación de combos y menús  
- Registro de productos más y menos vendidos  
- Control total del inventario  

---

## Alcance del proyecto

El sistema contempla el diseño y desarrollo de una plataforma móvil enfocada en la gestión de pedidos y pagos en entornos escolares.

### Alumno / Docente
- Visualización de menú digital  
- Realización de pedidos anticipados  
- Uso de billetera virtual  
- Historial de compras  
- Sistema de reputación  

### Padres / Tutores
- Carga remota de saldo  
- Visualización de gastos en tiempo real  
- Control del consumo  
- Restricciones alimentarias  

### Buffet (Administrador)
- Gestión de stock y precios  
- Validación de pedidos mediante código  
- Panel de preparación organizado  
- Registro de incidencias  
- Reportes de ventas  

### Seguridad del sistema
- Códigos de retiro de un solo uso  
- Notificaciones automáticas  
- Alertas por saldo bajo o compras relevantes
